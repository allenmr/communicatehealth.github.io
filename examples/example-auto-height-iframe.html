Auto-Height iframe

All the Accessibile Interactions examples contain auto-height-iframe.js, which sends cross-domain friendly messages about the content's height to a parent window. Code in the examples below listens for the messages and adjusts the frames' heights accordinly.

<iframe id="accordion" width="100%" src="//communicatehealth.github.io/examples/accordion/" title="Accordion" scrolling="no" style="border:none;outline:1px solid black">
  <p>Your browser does not support iframes.</p>
</iframe>
<script>(function(f){f=document.getElementById("accordion");window.addEventListener('message',function(e){if(f.contentWindow===e.source){f.height=e.data;}});}());</script>
