Rich Select Menu

View isolated HTML

Presents a menu that appears like a set of select boxes. Selecting one of the boxes navigates to that selection. Keyboard controls match those of a normal select element. Adapted from HealthyPeople.gov's Health Disparities widget.

First set of selections:

  • Accordion: The accordion is a graphical control element comprising a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "stretched" to reveal the content associated with that item.
  • Dropdown Menu: In a graphical drop-down menu, clicking on an item (text word, button or icon ) causes a list of new items to appear below that item.
  • Off-canvas Navigation: Off-canvas menus are positioned outside of the viewport and slide in when activated.
